![자격증 챌린지 어플이 필요한 이유](https://user-images.githubusercontent.com/40999597/164878996-53d7da2f-6e33-400f-9367-b376f32ad66f.png)

1. 사람들과 공부인증 기능으로 매일 의지를 다지기!
2. 인증률에 따른 포인트 환급 기능!
